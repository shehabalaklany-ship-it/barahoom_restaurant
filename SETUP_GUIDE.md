# 🍖 دليل شامل لموقع مطعم برهوم للكباب

## 📱 كيفية فتح المشروع في Visual Studio Code

### الخطوة 1: تثبيت المتطلبات
تأكد من تثبيت:
- **Node.js** (الإصدار 16 أو أحدث) - [تحميل من هنا](https://nodejs.org/)
- **Visual Studio Code** - [تحميل من هنا](https://code.visualstudio.com/)
- **pnpm** (مدير الحزم) - شغّل في Terminal:
  ```bash
  npm install -g pnpm
  ```

### الخطوة 2: فتح المشروع
1. افتح **Visual Studio Code**
2. اذهب إلى **File** → **Open Folder**
3. اختر مجلد `barahoom_restaurant` الذي حملته
4. انتظر حتى ينتهي من تحميل الملفات

### الخطوة 3: تثبيت الحزم
1. افتح **Terminal** في VS Code (Ctrl + `)
2. شغّل الأمر:
   ```bash
   pnpm install
   ```
3. انتظر حتى ينتهي التثبيت

### الخطوة 4: تشغيل الموقع محلياً
شغّل الأمر:
```bash
pnpm dev
```

ستظهر رسالة مثل:
```
➜  Local:   http://localhost:3000/
```

افتح هذا الرابط في المتصفح لرؤية الموقع!

## 🔧 كيفية تعديل الموقع

### تغيير الألوان
1. افتح `client/src/index.css`
2. ابحث عن `:root {`
3. عدّل قيم الألوان:
   ```css
   --accent: #d4af37;  /* غيّر هذا إلى اللون الذي تريده */
   ```

### تعديل النصوص
1. افتح `client/src/pages/Home.tsx`
2. ابحث عن النص الذي تريد تغييره
3. عدّله مباشرة وسيتحدث الموقع تلقائياً

### إضافة صور جديدة
1. احفظ الصورة في `/home/ubuntu/webdev-static-assets/`
2. استخدم الرابط المخصص في الكود:
   ```jsx
   <img src="/manus-storage/image_name.png" alt="الوصف" />
   ```
