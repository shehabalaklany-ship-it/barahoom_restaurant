## 🌐 الجزء الثاني: طرق النشر المجانية

### **الطريقة 1: Netlify (الأفضل والأسهل)** ⭐⭐⭐⭐⭐

**المميزات:**
- مجاني تماماً
- سريع جداً
- دعم HTTPS تلقائي
- سهل جداً

**الخطوات:**

1. **إنشاء حساب:**
   - اذهب إلى https://www.netlify.com
   - اضغط "Sign up"
   - اختر "GitHub" أو "Email"

2. **ربط المشروع:**
   - اضغط "New site from Git"
   - اختر GitHub
   - اختر مستودع المشروع
   - اضغط "Deploy"

3. **إعدادات البناء:**
   - Build command: `pnpm build`
   - Publish directory: `dist`

4. **انتظر 2-3 دقائق:**
   - سيظهر رابط مثل: `https://your-site.netlify.app`

---

### **الطريقة 2: Vercel** ⭐⭐⭐⭐⭐

**المميزات:**
- مجاني تماماً
- سريع جداً
- سهل جداً

**الخطوات:**

1. **إنشاء حساب:**
   - اذهب إلى https://vercel.com
   - اضغط "Sign Up"
   - اختر GitHub

2. **استيراد المشروع:**

   - Build Command: `pnpm build`

4. **انتظر:**
   - سيظهر رابط مثل: `https://your-project.vercel.app`

### **الطريقة 3: GitHub Pages** ⭐⭐⭐

**المميزات:**
- مجاني تماماً
- آمن جداً
- مدعوم من GitHub

**الخطوات:**

1. **إنشاء مستودع:**
   - اذهب إلى https://github.com

   - اختر Pages

   - اضغط Save

4. **الرابط:**
   - `https://username.github.io`

## 🔍 الجزء الثالث: تحسين الـ SEO (ظهور أول في Google)

### **المرحلة الأولى: تحسين الموقع نفسه (On-Page SEO)**

#### 1️⃣ **تحسين العنوان والوصف**

**تعديل ملف `client/index.html`:**

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  
  <!-- العنوان الرئيسي (مهم جداً!) -->
  <title>مطعم برهوم للكباب - أفضل مطعم كباب في صنعاء | شارع الخمسين والثلاثين</title>
  
  <!-- الوصف (يظهر تحت الرابط في Google) -->
  <meta name="description" content="مطعم برهوم للكباب البلدي والمشويات الشامية - أفضل مطعم كباب في صنعاء. فروع في شارع الخمسين والثلاثين. طعام طازة وخدمة ممتازة. اطلب الآن!">
  
  <!-- الكلمات المفتاحية -->
  <meta name="keywords" content="مطعم كباب, كباب برهوم, مطعم برهوم, كباب صنعاء, مشويات, كباب البلدي, مطاعم صنعاء">
  
  <!-- معلومات الموقع -->
  <meta name="author" content="مطعم برهوم">
  <meta name="robots" content="index, follow">
  
  <!-- Open Graph (لـ Facebook و WhatsApp) -->
  <meta property="og:title" content="مطعم برهوم للكباب - أفضل مطعم في صنعاء">
  <meta property="og:description" content="استمتع بأفضل الكباب والمشويات الشامية في مطعم برهوم">
  <meta property="og:image" content="https://your-domain.com/logo.png">
  <meta property="og:url" content="https://your-domain.com">
  <meta property="og:type" content="business.business">
  
  <!-- Twitter Card -->
  <meta name="twitter:card" content="summary_large_image">
  <meta name="twitter:title" content="مطعم برهوم للكباب">
  <meta name="twitter:description" content="أفضل مطعم كباب في صنعاء">
  
  <!-- Favicon -->
  <link rel="icon" type="image/png" href="/logo.png">
  
  <!-- Canonical URL -->
  <link rel="canonical" href="https://your-domain.com">
</head>
```

---

#### 2️⃣ **إضافة Schema Markup (البيانات المنظمة)**

**أضف هذا في `client/index.html` داخل `<head>`:**

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org/",
  "@type": "Restaurant",
  "name": "مطعم برهوم للكباب",
  "image": "https://your-domain.com/logo.png",
  "description": "مطعم برهوم للكباب البلدي والمشويات الشامية",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "شارع الخمسين",
    "addressLocality": "صنعاء",
    "addressCountry": "YE"
  },
  "telephone": "+967123456789",
  "url": "https://your-domain.com",
  "priceRange": "$$$",
  "cuisine": "Middle Eastern",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "150"
  }
}
</script>
```

#### 4️⃣ **إضافة Sitemap و Robots.txt**

**أنشئ ملف `client/public/sitemap.xml`:**

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://your-domain.com/</loc>
    <lastmod>2026-07-19</lastmod>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://your-domain.com/#menu</loc>
    <lastmod>2026-07-19</lastmod>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://your-domain.com/#branches</loc>
    <lastmod>2026-07-19</lastmod>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://your-domain.com/#location</loc>
    <lastmod>2026-07-19</lastmod>
    <priority>0.8</priority>
  </url>
</urlset>
```

**أنشئ ملف `client/public/robots.txt`:**

```
User-agent: *
Allow: /
Disallow: /admin

Sitemap: https://your-domain.com/sitemap.xml
```
