# 📚 دليل شامل لمشروع مطعم برهوم للكباب

## 🎯 الخطوة 1: فتح المشروع في Visual Studio Code

### المتطلبات الأساسية:
1. **Node.js** - [تحميل من هنا](https://nodejs.org/)
2. **Visual Studio Code** - [تحميل من هنا](https://code.visualstudio.com/)
3. **pnpm** (مدير الحزم) - شغّل في Terminal:
   ```bash
   npm install -g pnpm
   ```

### خطوات الفتح:

#### الخطوة 1: افتح VS Code
```
اضغط على File → Open Folder
```

#### الخطوة 2: اختر مجلد المشروع
```
ابحث عن مجلد "barahoom_restaurant" الذي حملته
اضغط على "Open"
```

#### الخطوة 3: افتح Terminal
```
اضغط Ctrl + ` (أو من القائمة: Terminal → New Terminal)
```

#### الخطوة 4: ثبّت الحزم
```bash
pnpm install
```
(انتظر حتى ينتهي - قد يستغرق دقيقة أو دقيقتين)

#### الخطوة 5: شغّل الموقع
```bash
pnpm dev
```

#### الخطوة 6: افتح في المتصفح
```
انسخ هذا الرابط: http://localhost:3000/
الصقه في متصفحك
```

## 🛠️ كيفية التعديل على الملفات

### تغيير النصوص:
1. افتح `client/src/pages/Home.tsx`
2. ابحث عن النص الذي تريد تغييره
3. عدّله واحفظ الملف (Ctrl + S)
4. الموقع سيتحدث تلقائياً!

### تغيير الألوان:
1. افتح `client/src/index.css`
2. ابحث عن `:root {`
3. عدّل قيم الألوان:
```css
--accent: #d4af37;  /* غيّر هذا إلى اللون الذي تريده */
```

### إضافة صفحة جديدة:
1. أنشئ ملف جديد في `client/src/pages/`
2. سمّه `NewPage.tsx`
3. أضفه إلى التوجيه في `App.tsx`
